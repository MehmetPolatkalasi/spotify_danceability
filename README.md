# Spotify Danceability Prediction & Playlist Generator

Collaborative machine learning project developed with Deniz Can Elci.

## Project Overview

This project predicts Spotify track danceability scores using machine learning models and generates personalized playlists through Spotify API integration.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Spotify API

## Features

- Danceability prediction
- Regression modeling
- Gradient boosting techniques
- Feature engineering
- Playlist generation
- Model evaluation

## Dataset

Approximately 140,000 Spotify tracks were used for training and evaluation.

## Results

Achieved RMSE below 0.10 during model evaluation.
